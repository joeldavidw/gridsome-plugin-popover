# gridsome-plugin-popover

> Add Vue.js popover to Gridsome - [Popover](https://github.com/euvl/vue-js-popover)

## Installation
```bash
yarn add gridsome-plugin-popover # or
npm i gridsome-plugin-popover
```

## Usage
```bash
module.exports = {
  plugins: [
    {
      use: 'gridsome-plugin-popover'
    }
  ]
}
```
